# 오행 세력 분석 UI 컴포넌트 구현

## 📅 작업일: 2025-12-16

## 📋 요청 사항

1. **오행 세력표 데이터화** - Phase 1에서 계산한 가중치 적용 오행 점수를 화면에 표시
2. **신강/신약 결과 컴포넌트** - Phase 2 판정 결과를 시각적으로 보여주는 UI
3. **오행표 가중치 토글** - 기본 오행 숫자 ↔ 가중치 적용 점수 전환 버튼

---

## 🔧 구현 내용

### 생성된 파일

#### `/components/OhaengForceDisplay.tsx`

새로운 오행 세력 분석 컴포넌트

**주요 기능:**

| 기능 | 설명 |
|------|------|
| **기본/가중치 토글** | 버튼 클릭으로 기본 숫자 ↔ 가중치 적용 점수 전환 |
| **별모양 오행표** | 기존과 동일한 오각형 다이어그램 (점수에 따라 원 크기 변화) |
| **바 차트** | 오행별 점수와 백분율을 막대 그래프로 표시 |
| **합/충 표시** | 가중치 적용 시 발견된 합/충 상호작용 표시 |
| **신강/신약 판정** | 득령/득지/득세, 지수, 등급 등 상세 정보 표시 |

```

│                 │                 │
│   별모양        │   바 차트       │
│   오행표        │   (백분율)      │
│                 │                 │
└─────────────────┴─────────────────┘
```

### 3. 신강/신약 판정 (가중치 적용 시)

```

│  │득령│ │득지│ │득세│              │

│─────────────────────────────────────│
```

## 🎨 디자인 특징

1. **색상 체계**
   - 목(木): 초록색 `#00B050`
   - 화(火): 빨간색 `#FF0000`
   - 토(土): 노란색 `#FEC100`
   - 금(金): 회색 `#9CA3AF`
   - 수(水): 검은색 `#000000`

2. **신강/신약 등급별 색상**
   - 태왕 🔥🔥: 빨간색
   - 신강 🔥: 주황색
   - 중화 ⚖️: 초록색
   - 신약 💧: 파란색
   - 태약 💧💧: 남색

3. **애니메이션**
   - 토글 전환 시 부드러운 트랜지션
   - 바 차트 애니메이션 (500ms)
